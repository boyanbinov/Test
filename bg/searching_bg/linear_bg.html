int search(int arr[], int n, int x) { int i; for (i = 0; i < n; i++) if (arr[i] == x) return i; return -1; }
При линейното търсене масивът не е нужно да бъде сортиран. При него последователно се обхожда всеки елемент от масива, докато търсеният елемент не бъде намерен.
Линейното търсене има средна скорост от O(n). Най-лошият случай има скорост от O(n), а в най-добрият случай има скорост от O(1).
