void heapify(int arr[], int n, int i) { int largest = i; int l = 2 * i + 1; int r = 2 * i + 2; if (l < n && arr[l] > arr[largest]) largest = l; if (r < n && arr[r] > arr[largest]) largest = r; if (largest != i) { swap(arr[i], arr[largest]); heapify(arr, n, largest); } } void heapSort(int arr[], int n) { for (int i = n / 2 - 1; i >= 0; i--) heapify(arr, n, i); for (int i = n - 1; i > 0; i--) { swap(arr[0], arr[i]); heapify(arr, i, 0); } }
При пирамидалното сортиране се използва самобалансиращо бинарно дърво за търсене (complete binary tree). Heap структура от данни - всички възли в дървото следват свойството, че те са по-големи от техните деца, т.е. най-големият елемент е в корена и двата му дъщерни елементи са по-малки от корена. Такъв heap се нарича max-heap. Ако вместо това всички възли са по-малки от техните деца, това се нарича min-heap. Първата стъпка при пирамидалното сортиране е дървото да се превърне в max heap. Функция heapify отговаря за размяната на елементи в дървото. Когато изискванията за max heap са изпълнени пирамидалното сортиране прави размяна на корена на дървото и последния елемент в масива. Когато размяната се осъществи премахваме елемента, който преди е бил корен от heap-а. След това пак се използва помощната функция heapify, за да имаме най-голямата стойност от масива, като корен. Този процес се повтаря, докато масивът не се сортира.
Пирамидалното сортиране има скорост от O(n logn) във всички случаи най-добър, среден и най-лош.
