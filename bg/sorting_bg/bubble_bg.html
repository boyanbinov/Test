static void bubbleSort(int []arr) { int n = arr.Length; for (int i = 0; i < n - 1; i++) for (int j = 0; j < n - i - 1; j++) if (arr[j] > arr[j + 1]) { int temp = arr[j]; arr[j] = arr[j + 1]; arr[j + 1] = temp; } }

Взимаме първият елемент на масива и го сравняваме със следващия и разменяме стойностите им, ако първият е по-голям от втория. След това сравняваме вторият елемент с третия и пак разменяме, ако има нужда. Когато обхождането достигне края на масива започваме отново да сравняваме като пак взимаме първият елемент и сравняваме с втория и така нататък. В най-лошият случай когато масивът е обърнат на обратно, скоростта на алгоритъма според е O(n2). Средната му скорост също се измерва в O(n2). В най-добрият случай когато масивът е сортиран скоростта на алгоритъма в този случай се измерва в O(n).
